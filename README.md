# TP---Objetos
Trabajo practico de Objetos

Mi proyecto de objetos se orienta a una clase Pizarra, que dentro de ella, tiene una función de hacer operaciones y otra que hace que se dibuje sobre la “Pizarra” cuales son los comandos que se ejecutan en esa operación. En este caso, la calculadora recibe dos parámetros numéricos, y un parámetro que seria el operador, se hace llamado a la función hacerOperacion, y una vez que esta se ejecuta, haciendo el llamado a la función dibujarCalculadora, te imprime por consola esa operación.

Para este proyecto utilice la librería “Cli-diagram”, que te perimte hacer dibujos de diagramas a través de funciones. Con ella pude hacer una composición a la clase Pizarra y hacer el dibujo de la calculadora, en la consola.
